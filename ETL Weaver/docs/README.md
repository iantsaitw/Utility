# 📸 Media Assets & Screenshots

To maintain consistency in the main README, please follow these naming conventions for screenshots:

### 🖼️ Standard Filenames
- main_ui.png: The primary ETL conversion interface.
- settings_ui.png: The side-by-side settings and font configuration.
- conversion_log.png: Real-time feedback window during ETL processing.

### 📝 Guidelines
- **Format**: Preferred .png.
- **Update**: Replacing these files automatically updates the visual showcase in the root project description.
